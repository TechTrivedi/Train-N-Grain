**Train N Grain** is a full-stack web application designed for personalized workout generation, intelligent nutrition planning, macro tracking, and real-time fitness analytics. Built for fitness enthusiasts looking for tailored guidance on their physical transformation journey.

🌐 **Live Website**: [https://train-n-grain.vercel.app](https://train-n-grain.vercel.app)

---

## 🌟 Key Features

- **🏋️‍♂️ Custom Workout Generator**: Tailored exercise routines based on fitness goals (muscle gain, fat loss, endurance) and user experience level.
- **🥗 Smart Nutrition & Macro Planner**: Calculates daily caloric needs and macronutrient distribution (proteins, carbs, fats) tailored to individual fitness targets.
- **🔐 Firebase User Authentication**: Secure user login, registration, and persistent session management powered by Firebase Auth.
- **📊 Real-time User Profile & Progress Tracking**: Interactive profile dashboard for tracking workout consistency and nutrition stats over time.
- **📱 Responsive Glassmorphic Design**: Modern dark theme UI with smooth micro-animations, dynamic cards, and custom CSS styling across mobile, tablet, and desktop devices.
- **⚡ Serverless Integration**: Lightweight API architecture for fast loading times and instant data synchronization.

---

## 🛠️ Tech Stack

- **Frontend**: JavaScript (ES6+ Modules), HTML5, Vanilla CSS3 (Custom Glassmorphism & HSL variables)
- **Backend & Database**: Firebase Authentication, Firestore Database, Serverless Node.js APIs
- **Hosting & Deployment**: Vercel (`train-n-grain.vercel.app`)

## 🚀 Getting Started Locally

### Prerequisites
- Modern web browser (Chrome, Firefox, Edge, Safari)
- Node.js (v18.x or later)

### Installation & Local Run

1. **Clone the repository**:
   ```bash
   git clone https://github.com/TechTrivedi/Train-N-Grain.git
   cd Train-N-Grain
   ```

2. **Serve locally**:
   - You can use VS Code **Live Server** extension or run a simple local server:
   ```bash
   npx serve .
   ```

3. **Open in browser**:
   Navigate to `http://localhost:3000` or `http://127.0.0.1:5500`.
